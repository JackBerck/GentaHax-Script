# GentaHax-Script

Kumpulan script Lua untuk game Growtopia.

## Deskripsi

Genta Hax adalah perangkat lunak pihak ketiga atau script modifikasi tidak resmi yang digunakan untuk game Growtopia. Alat ini biasanya digunakan oleh pemain untuk memodifikasi gameplay, menjalankan script otomatis (Lua), hingga mengakses fitur kecurangan (cheat) guna mempermudah permainan.

> **Peringatan:** Aplikasi semacam ini melanggar ketentuan layanan (ToS) resmi dari pengembang game dan berisiko tinggi menyebabkan akun Anda terkena banned (pemblokiran). Gunakan dengan bijak dan risiko ditanggung sendiri.

## Daftar Script

Berikut adalah daftar script yang tersedia saat ini:

- **AutoFindWorld.lua**: Script otomatis untuk mencari world.
- **ReduceCrashGentaAndroid.lua**: Script untuk membantu mengurangi crash saat menggunakan Genta di Android.
- **private-scripts/**: Script khusus (PRIVATE).

Repository ini akan terus diupdate seiring berjalannya waktu dengan penambahan script-script baru.
