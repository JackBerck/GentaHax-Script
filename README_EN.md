# GentaHax-Script

A collection of Lua scripts for the game Growtopia.

## Description

Genta Hax is a third-party software or unofficial modification script used for the game Growtopia. This tool is typically used by players to modify gameplay, run automated scripts (Lua), and access cheating features to make the game easier.

> **Warning:** This kind of application violates the official Terms of Service (ToS) of the game developer and carries a high risk of your account being banned. Use at your own risk.

## Script List

Current available scripts:

- **AutoFindWorld.lua**: Automatically searches for worlds.
- **ReduceCrashGentaAndroid.lua**: Helps reduce crashes when using Genta on Android.
- **private-scripts/**: Private scripts (not public).

This repository will be continuously updated with new scripts over time.
